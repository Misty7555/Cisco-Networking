# 🌐 Cisco Networking Labs (Packet Tracer)

This repository contains hands-on networking labs created using Cisco Packet Tracer.  
The labs cover networking fundamentals, router and switch configuration, VLANs, subnetting, dynamic routing protocols, and network security concepts.

---

# 📁 Labs Included

## 🔹 Lab 1: Basic Configuration

- Configure hostname, passwords, and basic device settings
- Assign IP addresses to devices
- Verify connectivity using `ping`

📌 **Concepts Covered:**
- IP Addressing
- Basic Router/Switch Configuration
- Connectivity Testing

---

## 🔹 Lab 2: Enable Telnet

- Configure remote login using Telnet
- Secure remote access using passwords
- Test connectivity remotely

📌 **Concepts Covered:**
- Telnet Protocol
- Remote Device Access
- Basic Network Security

---

## 🔹 Lab 3: Router Password & Banner Configuration

---

# 🎯 Objective

The objective of these labs is to build a strong understanding of computer networking through practical implementation of routing, switching, subnetting, VLANs, and dynamic routing protocols used in real-world networks.

---

# 🚀 How to Use

1. Download the `.pkt` files
2. Open them using Cisco Packet Tracer
3. Explore configurations
4. Test connectivity and routing
