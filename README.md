# 🌐 Cisco Networking Labs (Packet Tracer)

This repository contains my hands-on networking labs created using Cisco Packet Tracer.
These labs demonstrate fundamental networking concepts including IP addressing, Telnet configuration, and router security.

---

## 📁 Labs Included

### 🔹 Lab 1: Basic Configuration

* Configure hostname, passwords, and basic device settings
* Assign IP addresses to devices
* Verify connectivity using `ping`

📌 **Concepts Covered:**

* IP Addressing
* Basic Router/Switch Configuration
* Connectivity Testing

---

### 🔹 Lab 2: Enable Telnet

* Configure remote login using Telnet
* Secure access using passwords
* Test remote connectivity

📌 **Concepts Covered:**

* Telnet Protocol
* Remote Device Access
* Network Security Basics

---

### 🔹 Lab 3: Router Password & Banner Configuration

* Set console and enable passwords
* Configure login banner
* Improve basic router security

📌 **Concepts Covered:**

* Device Security
* Password Encryption
* MOTD Banner

---

## 🛠️ Tools Used

* Cisco Packet Tracer
* CLI (Command Line Interface)

---

## 🎯 Objective

The goal of these labs is to build a strong foundation in networking by implementing real-world configurations and understanding how devices communicate in a network.

---

## 🚀 How to Use

1. Download `.pkt` files
2. Open using Cisco Packet Tracer
3. Explore configurations and test connectivity
