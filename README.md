# 📌 Overview

This repository contains a collection of hands-on Cisco networking labs designed and implemented using **Cisco Packet Tracer**.

The project demonstrates practical knowledge of:

- Router & Switch Configuration
- VLANs & Inter-VLAN Communication
- Dynamic Routing Protocols
- Subnetting & VLSM
- DHCP Configuration
- ACL (Access Control Lists)
- OSPF, RIP & EIGRP Routing
- Network Security Fundamentals
- End-to-End Connectivity Troubleshooting

These labs simulate real-world enterprise networking scenarios and are aimed at strengthening practical networking skills through configuration, testing, and troubleshooting.
